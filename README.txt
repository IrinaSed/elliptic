Программа принимает файл в формате:
1) тип:
	<number>       - char != 2,3
	`2S`           - char = 2, Supersingular
	`2N`           - char = 2, notSupersingular
2) поле:
	1              - char != 2,3
	<number>       - char = 2
3) параметры кривой:
	3.1) <number>  - a
	3.2) <number>  - b,
	3.3) <number>  - c, если char = 2, иначе пункт 5
4) <number>        - если char = 2, иначе пункт 5
                     многочлен, например x^4 + x + 1 = 0
                     записывается в виде 0b10011 или 19
5) Список задач(tasks):
	`c` <x1> <y1> <x2> <y2>
	`у` <x1> <y1> <number>


Чтобы проверить программу на своих тестовых файлах можно:
1)-вариант:
    положить эти файлы в папку tests
    перейти в директорию `eliptic`
    и запустить программу python3 main.py
2)-вариант
    перейти в директорию `eliptic`
    и запустить программу(с аргументами, то есть указать пути до файлов):
    python3 main.py file1, file2,...

Результат можно посмотреть в папочке `tests_answer`
